## 🎨 Design System

### Colors
- **Primary Blue**: #2563eb (All CTAs, hover states)
- **Accents**: Green (success), Red (warnings), Orange (actions)
- **Text**: Gray-900 (primary), Gray-600 (secondary)
- **Backgrounds**: White (base), Blue-50 (sections), Gray-900 (footer)

### Typography
- H1: 56px (desktop), 40px (mobile)
- H2: 36px (sections)
- H3: 20px (cards)
- Body: 16px
- Caption: 14px

### Spacing
- Section padding: 80px
- Card gap: 24px
- Component gap: 16px

### Components Used
- Cards with hover effects
- Icons from Lucide React
- Gradient backgrounds (subtle)
- Details/Summary (native HTML)
- Responsive grid layouts

## 💪 What Makes This Landing Strong

### 1. Clarity
- No jargon, no hype
- User understands OnKey in 5 seconds
- Every section has clear purpose

### 2. Specificity
- "Ahorra 10 horas" not "Ahorra tiempo"
- "45 propiedades" not "mucho"
- "14 días gratis" not "Try free"

### 3. Problem-First
- Doesn't start with product features
- Starts with user pain
- Then shows how we solve it

### 4. Low Friction
- "Sin tarjeta de crédito" mentioned early
- Multiple CTAs (not just one)
- Easy navigation
- Mobile friendly

### 5. Social Proof
- 3 real testimonials (not generic praise)
- Specific numbers (propiedades, horas ahorradas)
- Trust signals (datos seguros, respaldos)

### 6. Visual Hierarchy
- Big headline → Easy to scan
- Color coding (problems=red, solutions=blue)
- Whitespace to reduce cognitive load

## ⚠️ What NOT to Do

```
❌ Change copy without data/testing
❌ Add too many sections (info overload)
❌ Use generic stock photos
❌ Hide pricing
❌ Misleading claims ("100x your income")
❌ Technical jargon
❌ Multiple CTAs doing different things
❌ Slow load times (>3s)
❌ No mobile optimization
❌ Ignore analytics
```
