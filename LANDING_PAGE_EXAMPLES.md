# 🔍 OnKey Landing - Copy Examples: Good vs Bad

## Hero Section Comparison

### ❌ BAD Headline (Generic)
```
"The Best Software for Real Estate Management"

Problems:
- Generic ("best" is empty)
- Not specific to immobiliarios
- Boring, sounds like every SaaS
- User: "So what? I don't care."
```

### ✅ GOOD Headline (OnKey's)
```
"Un sistema para todo tu negocio inmobiliario"

Why it works:
- Specific ("todo tu negocio")
- Targets exact audience (inmobiliario)
- Clear promise (one system)
- User: "Oh! They get me."
```

---

### ❌ BAD Subheadline
```
"Streamline your workflow with powerful features"

Problems:
- Vague (what features?)
- Buzzwords (streamline, powerful)
- No benefit (what do I actually get?)
- User: "...okay?"
```

### ✅ GOOD Subheadline (OnKey's)
```
"Propiedades, contratos, contactos y alertas en un mismo lugar. 
Ahorra 10 horas semanales en tareas administrativas."

Why it works:
- Specific (lists what's included)
- Concrete number (10 horas)
- Emotional benefit (tareas administrativas gone)
- User: "YES! This solves my problem!"
```

---

## Problem Section Comparison

### ❌ BAD Problem Statement
```
"Managing multiple properties can be difficult"

Issues:
- Too soft (not relatable)
- "Can be" = wishy-washy
- Not specific
- User: "Yeah, but is this really that hard?"
```

### ✅ GOOD Problem Statement (OnKey's)
```
"Propiedades en Excel, contratos en carpetas, 
mensajes en WhatsApp... ningún único lugar de verdad."

Why it works:
- VERY specific (Excel, carpetas, WhatsApp)
- Situation recognizable (user nods: "That's exactly me!")
- Frustration level high (no single place)
- User: "OMG, they described my life perfectly"
```

---

### ❌ BAD Benefit Claim
```
"Save time with automation"

Problems:
- Too generic
- No number
- No context
- User: "How much time? I don't believe it."
```

### ✅ GOOD Benefit Claim (OnKey's)
```
"Pierdes horas en tareas administrativas"
"Tareas sin valor que roban tu tiempo"
[Later] "Ahorra 10 horas semanales"

Why it works:
- Emotion first (pierdes, roban)
- Specific number (10 horas)
- Specific context (administrativas, sin valor)
- User: "That's real. I believe it."
```

## CTA Copy Comparison

### ❌ BAD CTA Text
```
"Get Started"
```
Problems:
- Generic (used by every SaaS)
- No benefit stated
- No urgency
- Weak verb

### ❌ Also BAD
```
"Sign Up Now"
```
Problems:
- "Now" is artificial urgency
- "Sign Up" = commitment scary
- User: "Why am I committing?"

### ✅ GOOD CTA (OnKey's)
```
"Probar gratis 14 días"
```
Why it works:
- Action clear (Probar = try)
- Friction low (gratis)
- Time specific (14 días)
- No risk for user
- User: "I can try without risk"

### ✅ ALSO GOOD CTA (Alternative)
```
"Empezar ahora"
```
Why it works:
- Simple, direct
- "Empezar" = action
- "Ahora" = instant gratification
- Natural language
```

## Pricing Comparison

### ❌ BAD Pricing Copy
```
Pro Plan: $49/month

✓ Advanced features
✓ Priority support
✓ Custom reports
```
Problems:
- "Advanced" is vague
- Features don't translate to benefits
- User: "Why should I pay for this?"

### ✅ GOOD Pricing Copy (OnKey's)
```
Pro Plan: $49/mes
✓ Propiedades ilimitadas         ← scales
✓ Gestión completa de contactos  ← specific
✓ Dashboard avanzado              ← what you need
✓ Gestión de contratos           ← pain point solved
✓ Alertas inteligentes           ← main benefit
✓ Reportes personalizados        ← specific, usable
```
Why it works:
- Each benefit is specific
- Features translate to user outcomes
- "Ilimitadas" = growth potential
- "Inteligentes" = automation benefit

### ❌ BAD: Hiding Price
```
"Enterprise Plan: Contact Sales"
```
Problem: User assumes overpriced

### ✅ GOOD: Transparent about Enterprise
```
"Enterprise Plan: $199+/mes
✓ Todo en Pro
✓ Usuarios ilimitados
✓ Integraciones personalizadas"
```
Why it works:
- Price visible
- Transparente about what's included
- Clear upgrade path

---

### ❌ BAD: Over-the-top
```
"Stop drowning in paperwork! Your business 
is bleeding money! URGENT ACTION NEEDED!"
```

- Sounds like spam
- Doesn't trust reader's intelligence

```

- Respectful tone (not yelling)

## FAQ Copy Comparison

### ❌ BAD FAQ
```
Q: Do you have customer support?
A: Yes! 24/7 support available.

Problems:
- Too generic
- Doesn't address real concern
- "24/7" might not be true
- User: "That's not what I need to know"
```

### ✅ GOOD FAQ (OnKey's)
```
Q: ¿Tengo que pagar tarjeta para la prueba gratuita?
A: No. Los 14 días de prueba gratis no requieren 
tarjeta de crédito. Te das de alta, pruebas todo 
el sistema, y decides si quieres continuar.

Why it works:
- Addresses actual objection (not generic)
- Clear reassurance (no, no tarjeta)
- Concrete process (da de alta → prueba → decide)
- User: "Okay, my main worry is gone"
```

## Email Copy Comparison

```
Subject: OnKey is here!

Hi [Name],

OnKey is excited to announce our new platform! 
With advanced features and powerful capabilities, 
OnKey helps you manage your business better.

Check us out: [Link]

Best,
OnKey Team

Problems:
- Subject boring
- Generic opening
- No benefit stated
- Vague features
- Low CTR expected: <2%

```
Subject: Nunca más vencimientos olvidados 🚨

Hi [Name],

You know this feeling: A contract just vencido. 
A payment you missed. A document that expired.

That's what inspired OnKey.

We built it to prevent exactly that. 
One system for everything: propiedades, contratos, 
alertas, todo en un lugar.

Try it free for 14 days: [Link]

No credit card required.

— [Founder Name]

P.S. It took us 6 months to build. 
Come kick the tires. We'd love your feedback.

Benefits:
- Subject specific + emoji (higher open rate)
- Opens with problem they know
- Shows we understand (personal)
- Clear benefit stated
- CTA specific (14 días, gratis)
- Personal touch (founder P.S.)
- Expected CTR: 6-10%
```

## Key Takeaways

### What Makes Copy Convert

```
1. SPECIFIC > Generic
   "Propiedades, contratos, contactos" 
   beats 
   "All your management needs"

2. BENEFITS > Features
   "Nunca olvidas vencimientos"
   beats
   "Sistema de alertas"

3. EMOTIONAL > Rational
   "Deja de perder dinero"
   beats
   "Reduce administrative costs"

4. CONCRETE > Abstract
   "10 horas semanales"
   beats
   "Ahorra tiempo"

5. NATURAL > Corporate
   "Sí, aceptamos Excel"
   beats
   "Compatible with legacy formats"

6. REASSURING > Pushy
   "Sin compromiso"
   beats
   "SIGN UP NOW LIMITED TIME"
```

## Testing These Variations

### A/B Test Framework
```
Good: "Deja de perder dinero"
Alternative: "Gestiona 10x más propiedades"

Measure: Hero CTA click rate
Minimum: 500 visitors per variant
Winner: Whatever decreases bounce rate most
```

### First Tests to Run (In Order)
```
1. Headline (biggest impact)
2. CTA button text
3. Testimonial authenticity
4. Email form fields (email only vs email+phone)
5. Feature emphasis (which 3 matter most)
```
