# 💬 OnKey Landing - Copywriting Deep Dive & Variaciones

## 📌 Principios Clave de Persuasión

### PAS Framework (Problem → Agitate → Solve)
```
1. PROBLEM: Identifica el dolor
2. AGITATE: Amplifica la urgencia/costo
3. SOLVE: Presenta la solución como inevitable
```

### Estructura del Copy Persuasivo
```
EMOTIONAL HOOK → VALIDATION → FEAR → CURIOSITY → SOLUTION → PROOF → ACTION
```

## 🎯 HERO SECTION - Variaciones de Headlines

### Opción 1 (Actual - Recomendada)
```
"Un sistema para todo tu negocio inmobiliario"
Subtítulo: "Propiedades, contratos, contactos y alertas en un mismo lugar. 
Ahorra 10 horas semanales en tareas administrativas."
```
**Por qué funciona:**
- Específico a inmobiliarias (no "business" genérico)
- Promesa clara de integración
- Número específico de ahorro de tiempo
- Directo sin ser agresivo

**CTR esperado:** 4-6%

---

### Opción 2 (Más emocional)
```
"Deja de perder dinero por desorganización"
Subtítulo: "Contratos olvidados, pagos perdidos, propiedades sin seguimiento. 
OnKey previene todo esto en un sistema único."
```
**Por qué podría funcionar:**
- Toca "fear" (dinero perdido)
- Especifica los problemas
- Promesa de prevención

**Riesgo:** Puede sentirse demasiado negativo, menos optimista

---

### Opción 3 (Beneficio directo)
```
"Gestiona 10x más propiedades. Con el mismo equipo."
Subtítulo: "Sin aumentar carga administrativa. OnKey automatiza lo tedioso 
para que crezcas sin complicaciones."
```
**Por qué podría funcionar:**
- Promesa de escalabilidad
- Habla a crecimiento (aspiración)
- Directo al pain point: "con el mismo equipo"

**CTR esperado:** 3-5%
**Riesgo:** Puede parecer demasiado agresivo ("10x")

---

### Opción 4 (Máxima claridad)
```
"Propiedades, contactos, contratos y alertas en un solo lugar"
Subtítulo: "Sin papeles, sin Excel, sin caos. Gestión inmobiliaria moderna 
en una plataforma. Prueba gratis 14 días."
```
**Por qué funciona:**
- Extremadamente clara
- Emocional (sin caos)
- Directa (prueba gratis)

## 💚 SOLUCIÓN - Copy Patterns

### Pattern 1: Problema-Solución (1:1)
```
Antes:
"Excel, carpetas, WhatsApp, mensajes, confusión"

Después:
"Un único dashboard. Todo centralizado. Una fuente de verdad."
```

### Pattern 2: Resultado-Oriented
```
No digas: "Dashboard con metrics"
Di: "Visualiza exactamente cuánto ganas cada mes"

No digas: "Sistema de alertas"
Di: "Recibe notificaciones inteligentes antes de que algo se olvide"
```

### Pattern 3: Emotionally Resonant
```
Weak: "OnKey integra tus datos"
Strong: "Respira profundo. Todo está ahí. Organizado. Seguro."

Weak: "Automatización de flujos"
Strong: "Las tareas tediosas se hacen solas. Tu equipo gana esa hora."
```

```

un pago, o un documento expira. Reacciona antes del problema.
```

### Feature Copy Guidelines
- ✅ Empieza con verbo acción (Dashboard, Gestiona, Crea, Visualiza)
- ✅ Incluye contexto (cuándo, dónde, quién)
- ✅ Beneficio concreto, no técnica
- ✅ Sub-benefits = "cómo se usa" + "por qué es útil"

## 🎤 TESTIMONIOS - Formula Ganadora

### Estructura Ganadora
```
1. QUIEN SOY (Nombre + Contexto)
   "Soy Martín, administrador de 45 propiedades"

2. PROBLEMA ESPECÍFICO (Qué te pasaba)
   "Andaba perdiendo vencimientos de contratos regularmente"

3. CÓMO LO DESCUBRISTE (Relevancia)
   "Probé OnKey sin espectativas, honestamente"

4. CAMBIO TANGIBLE (Resultado medible)
   "Desde día uno, no vuelvo a perder ni uno"

5. IMPACT EMOCIONAL/ECONÓMICO
   "Ahorré horas semanales y puedo enfocarse en cerrar más negocios"

6. RECOMENDACIÓN NATURAL
   "Para cualquier administrador, es indispensable"
```

### Ejemplo Completo (Bueno)
```
"Desde que uso OnKey, no vuelvo a perder un vencimiento. 
He ahorrado horas en administración y puedo enfocarse realmente 
en el negocio. Es indispensable."
— Martín Rodríguez, Administrador de 45 propiedades
```

**Por qué funciona:**
- Específico (no genérico)
- Resultado medible (vencimientos, horas)
- Aspiracional (enfocarse en negocios)
- Corto (fácil de leer)
- Creíble (contexto específico)

### Testimonios a Evitar
```
❌ "Amé OnKey. Cambió mi vida. 5 estrellas."
   Razón: Suena fake, genérico, exagerado

❌ "Es el mejor software que he visto jamás."
   Razón: Nadie habla así

❌ "Los chicos de OnKey son increíbles. Great team!"
   Razón: Sueña a que el founder escribió el testimonial
```

## 💰 PRICING - Copy Psychology

### Principios de Pricing Copy
```
1. TRANSPARENCIA: Muestra el precio, no lo escondas
2. JUSTIFICACIÓN: Por qué ese precio
3. COMPARACIÓN: Qué obtienes por eso
4. URGENCIA (sutil): Limitado, o popular choice
```

### Plan Copy Examples

#### FREE PLAN (Starter)
```
✅ Hasta 10 propiedades
✅ Gestión de contactos básica
✅ Dashboard simple
✅ Alertas básicas
```
**Psicología:** 
- "Suficiente para probar"
- "Pero no suficiente para escalar"
→ Incentiva upgrade

#### PRO PLAN (Popular)
```
🎯 Lo que la mayoría elige

✅ Propiedades ilimitadas
✅ Gestión completa de contactos
✅ Dashboard avanzado
✅ Gestión de contratos
✅ Alertas inteligentes
✅ Reportes personalizados
```
**Psicología:**
- "Popular choice" = social proof
- Más features (ilimitadas = escalable)
- "Avanzado" = aspiracional

#### ENTERPRISE PLAN
```
Para grandes portfolios

✅ Todo en Pro
✅ Usuarios ilimitados
✅ Integraciones personalizadas
✅ Soporte prioritario
✅ Capacitación del equipo
```
**Psicología:**
- Premium = Para los serios
- Customization = "Pensado para ti"
- Support prioritario = Tranquilidad

### Pricing Page CTA Copy

```
❌ "Get Started" (genérico, es 2005)
❌ "Sign Up Now" (urgencia fake)
❌ "Subscribe" (parece que cobrar, no probar)

✅ "Probar gratis 14 días" (específico, sin riesgo)
✅ "Empezar ahora" (neutral, directo)
✅ "Crear cuenta gratis" (honesto)
```

## 📋 FAQ - Best Copy

### Principios FAQ Copy
```
1. PREGUNTA: Debe ser la pregunta real del usuario, not what you think
2. RESPUESTA: Honesta, específica, corta
3. REASSURANCE: Quita una objeción o miedo
```

### FAQ Versions

#### Versión 1 (Actual)
```
Q: ¿Tengo que pagar tarjeta para la prueba gratuita?
A: No. Los 14 días de prueba gratis no requieren tarjeta de crédito. 
   Te das de alta, pruebas todo el sistema, y decides si quieres continuar.

Q: ¿Cuánto tiempo toma configurar OnKey?
A: Puedes tener tu primer propiedad registrada en menos de 5 minutos. 
   La migración de datos existentes es simple: importa desde Excel o 
   cuéntale a nuestro equipo.

Q: ¿Puedo traer mis datos actuales?
A: Sí. Aceptamos archivos Excel, CSV, o PDFs. Nuestro equipo puede ayudarte 
   a migrar tus datos sin perder nada en el proceso.

Q: ¿Está mi información segura?
A: Sí. Usamos encriptación de nivel empresa, respaldos automáticos diarios, 
   y cumplimos con estándares de privacidad. Tus datos son responsabilidad 
   seria para nosotros.

Q: ¿Puedo invitar a mi equipo?
A: Claro. Invita a tu equipo con control de permisos: algunos ven todo, 
   otros solo sus propiedades asignadas. Tú controlas quién ve qué.
```

**Tono:** Directo, honesto, reasegurador

---

#### Versión 2 (Más detalles)
```
Q: ¿Es realmente gratis o es un truco?
A: Completamente gratis. 14 días de acceso total a PRO. Sin tarjeta requerida. 
   Si al día 15 decides no continuar, nadie te molesta. Simplicidad.

Q: Tengo datos en Excel/Access/carpetas locales. ¿Los pierdo?
A: Nada se pierde. Nuestro equipo maneja las migraciones. Trae tus archivos, 
   nosotros lo estructuramos en OnKey. Es parte de la onboarding.

Q: ¿Necesito entrenar a mi equipo para usarlo?
A: OnKey es intuitivo (la mitad de tu equipo lo aprenderá en una sesión). 
   Ofrecemos capacitación gratis si lo necesitas.

Q: ¿Qué pasa si cancelo mi suscripción?
A: Acceso a tus datos por 30 días. Después, puedes descargar todo en Excel. 
   Cero retención forzada de data.

Q: ¿Tienen soporte?
A: Sí. Chat, email, y en plan PRO: soporte prioritario. 
   Respuesta garantizada en <4 horas.
```

**Tono:** Más comprensivo, reconoce objeciones reales

## 🎯 CTA BUTTON COPY - Microcopy

### Button Copy Variations (A/B Test)

**Hero Primary:**
```
❌ "Get Started" → Too generic
❌ "Sign Up" → Sounds like commitment
✅ "Probar gratis 14 días" → Specific, no risk
✅ "Empezar ahora" → Direct
```

```
❌ "Learn More" → Weak

```

**Pricing:**
```
❌ "Purchase" → Too transactional
✅ "Probar gratis 14 días" (Para Free/Pro)

```

**Final CTA:**
```
❌ "Subscribe Now" → Too salesy
✅ "Comenzar" → Simple directness
✅ "Acceso inmediato" → Benefit, urgency
```

## 📝 Red Flags in Copy (AVOID)

```
❌ "Revolutionary"
❌ "Game-changing"
❌ "The only solution"
❌ "Guaranteed results"
❌ "Unlimited potential"
❌ Tech jargon (blockchain, AI, cloud, etc.)
❌ Hyperbole ("10x your income")
❌ Unspecified benefits ("Save time" without number)
❌ Fake urgency ("Only 3 spots left!")
❌ Fake testimonials (too perfect language)
```

## ✅ Best Practices (Checklist)

### Headline Guidelines
- [ ] Clear in <5 seconds
- [ ] Specific benefit (not feature)
- [ ] No jargon
- [ ] 40-60 characters ideal
- [ ] Emotional resonance

### Subheadline Guidelines
- [ ] Clarifies main headline
- [ ] Specific number (time/money saved)
- [ ] Addresses main objection
- [ ] 60-100 characters

### CTA Copy Guidelines
- [ ] Action verb (Probar, Empezar, Ver, etc.)
- [ ] Benefit or outcome (gratis, ahora, sin compromiso)
- [ ] No hedge words (just, try, consider)
- [ ] First person pov ("Mi primera propiedad" not "Tu primera propiedad")

### Testimonial Guidelines
- [ ] Specific result (number, timeframe)
- [ ] Authentic voice (not marketing language)
- [ ] Name + title + context
- [ ] Benefit-focused (not feature-focused)
- [ ] Under 50 words ideal
